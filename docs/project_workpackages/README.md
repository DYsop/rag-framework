# Arbeitspakete (AP01–AP11)

Dieses Verzeichnis enthält die Arbeitspakete entlang der Kapitellogik der Masterarbeit. Jedes Arbeitspaket dient zugleich als Vorlage für ein GitHub-Issue (siehe .github/ISSUE_TEMPLATE/work_package.md) und enthält Ziel, Bezug zur Masterarbeit, Aufgaben, Akzeptanzkriterien, betroffene Dateien, erwartetes Ergebnis, Status, Priorität und Meilenstein.

## Nutzung mit GitHub Projects

Aus jedem Arbeitspaket kann ein Issue erstellt und einem GitHub-Project-Board zugeordnet werden. Empfohlene Spalten: Backlog, To Do, In Progress, Review, Done. Labels: work-package, experiment, bug.
